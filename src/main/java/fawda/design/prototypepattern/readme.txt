定义:　对象有一个原型，通过拷贝原型（字节码）创建新对象
    原型模式就是如何快速构建对象的方法总结，
    简单工厂讲getter、setter封装到某个方法中
    JDK提供实现Cloneable接口，实现快速赋值

    spring中：scope="prototype", scope="singleton"

适用场景：
    1、类初始化消耗资源较多
    2、ｎｅｗ产生一个对象需要非常繁琐的过程（数据准备、访问权限等）
    3、构造函数比较复杂
    4、循环提中产生大量对象时

类克隆机制：
    浅克隆：克隆地址
    深克隆：克隆新对象

    克隆方法，采用深克隆
        可能会出现单例被破坏的问题
        克隆与单例冲突,如果cope的是单例需要做以下操作：不实现Cloneable或者实现readResolve
