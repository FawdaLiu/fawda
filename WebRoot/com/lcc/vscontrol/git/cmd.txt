4、仓库，git可以管理的目录
echo "xxx" >> xx.txt //管道追加
//初始化
$ git init
//添加文件到仓库
$ git add xxx.txt
$ git commit -m "xxx" //-m:描述信息
//查看仓库状态
$ git status

$ git reset HEAD xx.txt//从仓库到工作区
$ git reset --hard commitId//返回到指定的版本
$ git checkout -- xx.txt//从工作区到本地

7、标签管理
标记，通过标签来回滚
//查看标签
$ git tag
//创建标签
$ git tag name
//指定提交信息
$ git tag -a name -m "comment"
//删除本地标签
$ git tag -d name
//删除远程标签
$ git push origin :refs/tags/标签名
//标签发布,推送标签
$ git push origin name
8、分支管理
//查看分支
$ git branch
//创建分支
$ git branch name
//切换分支
$ git checkout name
//合并分支
$ git merge 分支branchName
//删除分支
$ git branch -d name
//解决冲突后合并
git merge upstream/master
